V1-schemes-feedbacks

>>>

GET /api/v1/schemes/:scheme_id/feedbacks
List feedbacks

Examples

GET /api/v1/schemes/5/feedbacks.json
200
{
  "feedbacks": [
    {
      "id": 3,
      "score": 1,
      "description": "This is good scheme",
      "scheme_id": 5
    }
  ]
}
>>>

POST /api/v1/schemes/:scheme_id/feedbacks
Create a feedback

Examples

POST /api/v1/schemes/2/feedbacks.json
feedback[score]=1&feedback[description]=This+is+good+scheme
201
{
  "id": 1,
  "scheme_id": 2,
  "score": 1,
  "description": "This is good scheme",
  "approved": false,
  "created_at": "2014-03-27T10:32:44.373+00:00",
  "updated_at": "2014-03-27T10:32:44.373+00:00"
}

Params

Param name Description
score
required

Feedback score from 1 to 5


Value: Must be Integer.
description
required

Feedback text (description)


Value: Must be String
